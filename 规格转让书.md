## 关键设计约束

1. **单进程**：所有 Agent 在同一个 FastAPI 进程中
2. **事件驱动**：Agent 间通过 `event_bus` 通信，不直接调用
3. **只有编排 Agent 改状态**：功能 Agent 只产出数据和事件
4. **置信度 MVP 用固定阈值**：≥0.90 自动，0.75-0.89 标记，<0.75 复核
5. **LLM 切 DeepSeek-v4**：不用 Qwen3，已配好 API 客户端
6. **假阴性优先**：异常识别宁可误报不可漏报
